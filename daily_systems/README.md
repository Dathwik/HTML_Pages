# Daily Systems Dashboard

A personal productivity dashboard covering my full daily schedule, workout plan, and nutrition — hosted via GitHub Pages.

## 🔗 Live Site
👉 [dathwik.github.io/HTML_Pages/daily_systems](https://dathwik.github.io/HTML_Pages/daily_systems/)

## 📋 What's Inside
- **⚡ The Game Plan** — Full time-blocked free day schedule (5 AM – 10 PM)
- **🏋️ Workout Plan** — 6-day training split with exercises, sets & reps
- **🥗 Nutrition** — Meal plan with macros and daily totals

## 🛠️ Built With
- Pure HTML, CSS, JavaScript — zero dependencies
- Dark theme, interactive checklists, tabbed navigation
